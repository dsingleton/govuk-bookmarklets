GOV.UK Bookmarklets

  1. Gov-a-lytics Shows all the analytics values for a GOV.UK page, including format and rendering-app.
  2. goto-content-store Shows the content-store representation of the current URL
  3. goto-publisher Searches for this artifact in publisher, to see the published editions/editorial content.
  4. goto panopticon Searches for this artifact in panopticon. Useful for seeing what format something is in, or which application is responsible for publishing/displaying it.
  5. env-production Goes to the current page on the live/production environment
  6. env-dev Goes to the current page on your local dev environment
  7. env-integration - Goes to the current page on the integration environment
  8. PDF to Web - Use on a Whitehall PDF to go to the page of publication it was uploaded to.
  9. See Govspeak - Goes to the preview of a Smart Answer and shows its Govspeak markup. It only works on outcome pages.
  10. Highlight Components - Highlights where GOV.UK Components are used on the page.
